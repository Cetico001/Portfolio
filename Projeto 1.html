Chat de Atendimento Conectado a API do WPP

Descrição

Para automatizar atendimentos rápidos, construi um site para qualquer usuario pode utilizar o Wpp principal separadamente, e nossos clientes nos contatarem somente por um número, isso evita que cada um precisa de um número da empresa.

Para ter um primeiro atendimento, utilizei algumas mensagens prontas, que talvez o cliente consiga até mesmo resolver tais problemas sem precisar de um atendimento humano. Mas quando precisar, as mensagens vão encaminhar até o atendimento que ele precisa.

Tecnologias usadas

  • PHP - Linguagem principal do script
  • MySQLi - Extensão do PHP para conexão com MySQL
  • MySQL - Sistema de banco de dados relacional
  • Z-API - API de WhatsApp para envio de mensagens
  • Webhooks - Para recebimento de dados externos
  • cURL - Para fazer requisições HTTP à API do Z-API
  • JSON - Para parsing de dados de entrada e saída
  • Sistema de Logs - Com file_put_contents para arquivo de texto
  • Programação Orientada a Objetos - Uso de mysqli com métodos OOP
  • Programação Procedural - Funções personalizadas
  • Manipulação de Arquivos - Sistema de logging em arquivo .txt
  • Prepared Statements - Para segurança contra SQL Injection
  • APIs RESTful - Comunicação com serviços externos
  • State Machine - Sistema de estados (status 0, 1, 2, 3) para fluxo de conversa
  • JavaScript Vanilla (ES6+) - Linguagem principal do script
  • DOM API - Manipulação direta do Document Object Model
  • Fetch API - Para requisições AJAX modernas
  • FormData - Para envio de arquivos via formulário
  • Event Listeners - addEventListener para interatividade
  • Audio API - Controle de elementos "audio" com eventos play/pause
  • HTML5 Audio - Elementos "audio" com controles nativos
  • setInterval - Para atualização periódica de mensagens
  • File API - Upload e manipulação de arquivos via input type="file"
  • ES6 Features - const, let, arrow functions () => {}
  • Single Page Application (SPA) - Navegação sem recarregar a página
  • Sistema de Chat em Tempo Real - Com polling a cada 3 segundos
